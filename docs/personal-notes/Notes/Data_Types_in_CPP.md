# Data Types in C++

[[000_Index_DS_Algo]]

### Strings and `getline`

- To use strings use `#include<string>`.
- To read a single line from input at a time use

```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string str;
    getline(cin, str);

	// .. Other code here
}
```
