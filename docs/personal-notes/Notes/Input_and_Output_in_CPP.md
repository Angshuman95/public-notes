# Input and Output in C++

[[000_Index_DS_Algo]]

- Create two files `input.txt` and `output.txt`, for input and output.
- Within `main` use the following:

```cpp
int main() {
    freopen("input.txt", "r", stdin);
    freopen("output.txt", "w", stdout);

	// ... Other code here
}
```
